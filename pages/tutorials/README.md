<h1 align="center">Tutorials</h1>

Step-by-step tutorials organised by topic. Select a tutorial from the **Tutorials** menu or the table below.

Each page matches one of the five main course areas on the site. Subsections correspond to specific skills or video themes. Return to the [topic pages](/math_kickstarter/Topics/) for videos, quizzes, and exercises.

Complete exercises with **pen and paper** — no calculator or maths software — but you may use a standard formula sheet and these tutorials.

## How to use the tutorials

1. Open the tutorial page for the topic you are studying.
2. Work through the subsections in order, using pen and paper for the examples.
3. Check your work against the [topic page](/math_kickstarter/basic_arithmetic_and_equations/) exercises when you are ready.

Tutorial content is being added topic by topic. Pages marked with placeholders are still in preparation.
