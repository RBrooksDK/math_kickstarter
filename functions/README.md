#### Exercise 1

Find the domain of $f(x) = \dfrac{1}{x - 2}$.

??? answer "&nbsp;"

    All real numbers except $x = 2$, i.e. $\mathbb{R} \setminus \{2\}$.

#### Exercise 2

Find the inverse of $f(x) = 2x + 3$.

??? answer "&nbsp;"

    $f^{-1}(x) = \dfrac{x - 3}{2}$
