    Functions

    Work through the videos below in order. When you have watched them, take the quiz and try the exercises. A quiz score of at least 7 out of 10 suggests sufficient understanding.

    Learning objectives¶

    After working through this topic, you should be able to:

    • Explain domain, range, and the graph of a function
    • Recognise and work with linear, power, and exponential functions
    • Understand and find inverse functions where they exist

    Videos¶

    1. Functions and graphs¶

    See functions as input–output rules, identify domain and range, and read values from graphs.

    2. Common function types¶

    Recognise linear, power, and exponential functions and how each one grows or changes.

    3. Inverse functions¶

    Find inverse functions where they exist and connect exponentials with logarithms.

    Quiz¶

    When you have been through the videos, test your understanding with the quiz below. The quiz contains 10 questions. A score of at least 7 out of 10 suggests sufficient understanding. You can see your total score at the bottom of the page, as well as your score for each question. You can retake the quiz as many times as you like — results are not stored or assessed.

    Question 1

    Which of the following best describes the domain of a function?

    Score: 0 / 1

    Explanation: The domain is the set of all values that can be used as input to the function.

    Question 2

    What is the range of $f(x) = x^2$?

    Score: 0 / 1

    Explanation: Squaring any real number gives a result $\ge 0$, and every non-negative value is achieved (e.g. $f(0)=0$, $f(2)=4$). So the range is $[0, \infty)$.

    Question 3

    If $f(x) = 3x - 4$, what is $f(5)$?

    Score: 0 / 1

    Explanation: Substitute $x = 5$: $f(5) = 3(5) - 4 = 15 - 4 = 11$.

    Question 4

    Which of the following is a linear function?

    Score: 0 / 1

    Explanation: A linear function has the form $f(x) = ax + b$, where $a$ and $b$ are constants. Only $-2x + 7$ fits this form.

    Question 5

    The function $f(x) = 5x^3$ is an example of which type?

    Score: 0 / 1

    Explanation: A power function has the form $f(x) = ax^n$, where the variable appears as a base raised to a fixed exponent. Here $f(x) = 5x^3$.

    Question 6

    For $f(x) = 4 \cdot 3^x$, what are the initial value and the base?

    Score: 0 / 1

    Explanation: In $f(x) = b \cdot a^x$, $b$ is the initial value (when $x = 0$, $f(0) = b$) and $a$ is the base. So $b = 4$ and $a = 3$.

    Question 7

    Under what condition does a function have an inverse function?

    Score: 0 / 1

    Explanation: An inverse exists only when the function is one-to-one. If two different inputs give the same output, you cannot uniquely reverse the mapping.

    Question 8

    To find the inverse of $y = f(x)$ algebraically, which step comes first?

    Score: 0 / 1

    Explanation: Finding an inverse means reversing input and output. Swapping $x$ and $y$ reflects this reversal before solving for the new output variable.

    Question 9

    The graph of $f^{-1}$ is the reflection of the graph of $f$ across which line?

    Score: 0 / 1

    Explanation: Points $(a, b)$ on $f$ correspond to $(b, a)$ on $f^{-1}$. Swapping coordinates reflects the graph across the line $y = x$.

    Question 10

    Which statement correctly links logarithms and exponentials?

    Score: 0 / 1

    Explanation: If $y = a^x$, then $x = \log_a(y)$. The logarithm undoes exponentiation, so $\log_a$ and $a^x$ are inverse functions (for $a > 0$, $a \ne 1$).

    Exercises¶

    Exercise 1¶

    Find the domain and range of \(f(x) = \sqrt{x - 3}\).

     

    Domain: \(x \ge 3\), i.e. \([3, \infty)\).

    Range: \(f(x) \ge 0\), i.e. \([0, \infty)\).

    Exercise 2¶

    If \(g(x) = x^2 - 2x + 5\), evaluate:

    1. \(g(0)\)
    2. \(g(2)\)
    3. \(g(-1)\)
     
    1. \(g(0) = 5\)
    2. \(g(2) = 5\)
    3. \(g(-1) = 8\)

    Exercise 3¶

    Identify each function as linear, power, or exponential:

    1. \(f(x) = 7 - 3x\)
    2. \(h(x) = 2x^{0.5}\)
    3. \(p(x) = 100 \cdot 0.5^x\)
     
    1. Linear (\(f(x) = -3x + 7\))
    2. Power (\(h(x) = 2x^{1/2}\))
    3. Exponential (\(p(x) = 100 \cdot 0.5^x\))

    Exercise 4¶

    Rearrange each formula to make the indicated variable the subject:

    1. \(V = IR\) — solve for \(I\)
    2. \(y = mx + c\) — solve for \(x\)
    3. \(P = \dfrac{E}{t}\) — solve for \(t\)
     
    1. \(I = \dfrac{V}{R}\)
    2. \(x = \dfrac{y - c}{m}\)
    3. \(t = \dfrac{E}{P}\)

    Exercise 5¶

    For \(f(x) = 2x + 3\), find \(f(4)\) and \(f(a + 1)\).

     

    \(f(4) = 2(4) + 3 = 11\)

    \(f(a + 1) = 2(a + 1) + 3 = 2a + 5\)

    Exercise 6¶

    Find the inverse of \(f(x) = 4x - 9\). State the domain and range of both \(f\) and \(f^{-1}\).

     

    Swap \(x\) and \(y\): \(x = 4y - 9\)

    Solve for \(y\): \(y = \dfrac{x + 9}{4}\)

    So \(f^{-1}(x) = \dfrac{x + 9}{4}\)

    Domain of \(f\): all real numbers; range of \(f\): all real numbers.

    Domain of \(f^{-1}\): all real numbers; range of \(f^{-1}\): all real numbers.

    Exercise 7¶

    The power dissipated in a circuit is \(P = I^2 R\). Rearrange to express \(I\) in terms of \(P\) and \(R\), then evaluate \(I\) when \(P = 50\,\text{W}\) and \(R = 2\,\Omega\).

     

    \(I = \sqrt{\dfrac{P}{R}}\)

    \(I = \sqrt{\dfrac{50}{2}} = \sqrt{25} = 5\,\text{A}\)

    Exercise 8¶

    For \(f(x) = 3^x\) and \(g(x) = \log_3(x)\):

    1. Evaluate \(f(2)\)
    2. Evaluate \(g(27)\)
    3. Simplify \(g(f(x))\)
     
    1. \(f(2) = 3^2 = 9\)
    2. \(g(27) = \log_3(27) = 3\) because \(3^3 = 27\)
    3. \(g(f(x)) = \log_3(3^x) = x\) (for \(x > 0\))

    Exercise 9¶

    Sketch (on paper) the graphs of \(f(x) = 2x + 1\) and its inverse on the same axes. What line of symmetry do they share?

     

    \(f^{-1}(x) = \dfrac{x - 1}{2}\)

    The graphs are reflections of each other across the line \(y = x\).

    Exercise 10¶

    A population grows according to \(N(t) = 500 \cdot 2^t\), where \(t\) is time in years.

    1. What is the initial population?
    2. What is the population after 3 years?
    3. Rearrange the formula to express \(t\) in terms of \(N\) (use logarithms).
     
    1. Initial population: \(N(0) = 500 \cdot 2^0 = 500\)
    2. After 3 years: \(N(3) = 500 \cdot 2^3 = 500 \cdot 8 = 4000\)
    3. \(\dfrac{N}{500} = 2^t\), so \(t = \log_2\!\left(\dfrac{N}{500}\right)\)
