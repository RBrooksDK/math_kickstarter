<h1 align="center">Topics</h1>

Click a topic in the menu (or in the table below) to access videos, a self-check quiz, and exercises for that area. Work through the topics recommended to you after the pre-test, or review any topic you want to strengthen.

## What this course covers

| Topic | Subtopics |
| ----- | --------- |
| [Basic Arithmetic and Equations](/math_kickstarter/basic_arithmetic_and_equations/) | Order of operations, fractions, exponents and radicals, parentheses, equations, quadratic equations, rearranging formulae |
| [Functions](/math_kickstarter/functions/) | Functions and graphs, common function types, inverse functions |
| [Differential Calculus](/math_kickstarter/differential_calculus/) | Derivatives and tangents, differentiation rules |
| [Integral Calculus](/math_kickstarter/integral_calculus/) | Indefinite integrals and antiderivatives, definite integrals and areas |
| [Vectors in 2D](/math_kickstarter/vectors/) | Vectors in 2D, dot product, parametric equation of a line |

Some advanced techniques are **deliberately excluded** (for example the product, quotient, and chain rules for differentiation, and integration by substitution or partial fractions). The focus is on the core skills needed for engineering mathematics.

---

## Recommended order

If you need to work through several topics, we suggest the following order:

1. Basic Arithmetic and Equations
2. Functions
3. Differential Calculus
4. Integral Calculus
5. Vectors in 2D

Within each topic, watch the videos in the order they appear on the page before taking the quiz and working on the exercises.
